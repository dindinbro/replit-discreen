# Navigation et pages du site Discreen

## Pages principales

### Page d'accueil (/)
Recherche rapide avec barre de recherche centrale. Permet de lancer une recherche par email, pseudo, telephone, IP ou Discord ID.

### Page de recherche (/search)
Interface de recherche avancee avec :
- Recherche par criteres (interne)
- Recherche globale (LeakOSINT)
- Recherche Discord
- Recherche telephone
- GeoIP
- Decodeur NIR

### Tarifs (/pricing)
Affiche les differents plans d'abonnement et permet de s'abonner ou d'activer une cle de licence.

### Avis (/vouches)
Page des avis et evaluations laisses par les utilisateurs via le bot Discord.

### Contact (/contact)
Page de contact avec deux services payants :
- Demande de retrait de blacklist (50€)
- Demande d'informations (50€)

### Documentation (/documentation)
Conditions Generales d'Utilisation completes du service.

### Utilisateurs (/users)
Liste des utilisateurs inscrits sur la plateforme.

## Pages authentifiees

### Profil (/profile)
Parametres du compte utilisateur. Accessible via le menu deroulant sur le nom d'utilisateur dans l'en-tete.

### Cles API (/api-keys)
Gestion des cles API pour les utilisateurs du plan API. Permet de creer, lister et revoquer des cles.

### Administration (/admin)
Panneau d'administration reserve aux admins :
- Gestion des utilisateurs
- Gestion des cles de licence
- Gestion des categories
- Gestion des demandes de blacklist

## En-tete du site
L'en-tete affiche :
- Logo Discreen
- Liens de navigation (Home, Recherche, Avis, Utilisateurs, Prix, Contact)
- Bouton de changement de theme (clair/sombre)
- Indicateur de statut (en ligne/hors ligne)
- Menu utilisateur (si connecte) avec lien vers le profil et deconnexion

## Support
Pour toute aide, les utilisateurs peuvent :
1. Utiliser le chatbot (bulle en bas a droite)
2. Ouvrir un ticket sur Discord
3. Contacter via la page /contact
